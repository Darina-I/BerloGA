**Auth**
| Адрес запроса | Метод запроса | Параметры запроса | Необходимая роль | Описание |
|---------------|---------------|-------------------|------------------|----------|
| /auth/signup | POST | Данные о новом пользователе | user | Регистрация пользователя в системе |

**BoardGame**
| Адрес запроса | Метод запроса | Параметры запроса | Необходимая роль | Описание |
|---------------|---------------|-------------------|------------------|----------|
| /boardgames | GET | **cursor** (в какого индекса необходимо вывести значения), **limit** (сколько значений нужно вернуть), **filter** (необходимые фильтры для значений) | user | Получение списка настольных игр |
| /boardgames/{id} | GET | - | user | Получение информации о настольной игре по id |
| /boardgames | POST | Вся информация о новой настольной игре | admin | Добавление новой настольной игры |
| /boardgames/{id} | PATCH | Поля, которые необходимо изменить, и их новое значение | admin | Изменение информации о настольной игре по id |
| /boardgames/{id} | DELETE | - | admin | Удаление настольной игры по id|
| /boardgames/{id}/comments-threads | GET | **cursor** (в какого индекса необходимо вывести значения), **limit** (сколько значений нужно вернуть) | user | Получение всех комментариев по id настольной игры |
| /boardgames/{id}/comments-threads | POST | Необходимые поля для создания блока комментариев | user | Создание блока комментариев для настольной игры по ее id|

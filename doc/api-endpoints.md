👾**Auth**
| Адрес запроса | Метод запроса | Параметры запроса | Необходимая роль | Описание |
|---------------|---------------|-------------------|------------------|----------|
| /auth/register | POST | **nickname** (никнейм нового пользователя), **password** (пароль пользователя) | user | Регистрация пользователя в системе |
| /auth/login | POST | **nickname** (никнейм пользователя), **password** (пароль пользователя) | user | Вход пользователя в систему |
| /auth/refresh | POST | - | user | Обновление access_token |
| /auth/logout | POST | - | user | Выход пользователя из системы |

🎲**BoardGame**
| Адрес запроса | Метод запроса | Параметры запроса | Необходимая роль | Описание |
|---------------|---------------|-------------------|------------------|----------|
| /boardgames/{genres[]} | GET | - | user | Получение списка настольных игр с возможной фильтрацией по жанрам |
| /boardgames/{id} | GET | - | user | Получение информации о настольной игре по id |
| /boardgames | POST | Вся информация о новой настольной игре | admin | Добавление новой настольной игры |
| /boardgames/{id}/comments-blocks | GET | - | user | Получение всех комментариев по id настольной игры |
| /boardgames/{id}/comments-blocks | POST | **header** (название темы для блока комментариев) | user | Создание блока комментариев для настольной игры по ее id|
| /boardgames/{id}/genres | GET | - | user | Получение жанров настольной игры по id |
| /boardgames/{id}/genres | POST | **genreId** (id жанра) | admin | Сохранение жанра настольной игры по id |
| /boardgames/{id}/genres/{genreId} | DELETE | - | admin | Удаление жанра настольной игры по id |

🗿**User**
| Адрес запроса | Метод запроса | Параметры запроса | Необходимая роль | Описание |
|---------------|---------------|-------------------|------------------|----------|
| /users | GET | - | user | Получение информации о всех пользователях |
| /users/admin | GET | - | user | Получение информации о всех пользователях для админа|
| /users/{id} | GET | - | user | Получение информации о пользователе по id |
| /users/{id}/role | PATCH | - | admin | Изменение роли пользователя по id |

| /users/me/genres/{genre_id} | DELETE | - | user | Удаление любимого жанра текущего пользователя по переданному id |
| /users/me/boardgames | GET | - | user | Получение списка настольных игр, которые входят в библиотеку текущего пользователя |
| /users/me/boardgames/{game_id} | POST | Оценка новой настольной игры в библиотеке | user | Добавление настольной игры по ее id в библиотеку |
| /users/me/boardgames/{game_id} | DELETE | - | user | Удаление настольной игры из библиотеки по id|
